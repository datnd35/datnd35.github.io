## Mục tiêu bài viết

Dẫn dắt daily meeting với client không chỉ là "điểm danh ai làm gì". Người facilitator tốt cần:

- Kiểm soát **flow** của meeting
- Mời **đúng người, đúng lúc**
- Tránh khoảng **lặng và lạc đề**
- Tóm tắt rõ **action items**
- Kết thúc **gọn gàng, chuyên nghiệp**

Bài này cung cấp đầy đủ: **mindset, diagram flow, script mẫu, và tips thực chiến**.

## Meeting Script: Từ đầu đến cuối

### Mở đầu

```
"Good morning everyone. Let's get started."

"I'll help facilitate today's meeting and invite each owner
 to provide updates."

"We'll cover:
  (1) Task-by-task progress update
  (2) Today's plan
  (3) Blockers or support needed
  (4) Action items

 Let's keep it concise so we can all get back to work quickly."
```

### Mời từng người cập nhật

```
"[Tên], could you please give us a quick update on [task/topic]?"

  → "What was completed yesterday?"
  → "What are you working on today?"
  → "Do you have any blockers or need anything from the client side?"
```

### Tóm tắt sau mỗi người

```
"Got it. So [task] is [status], and the main point is [key info].
 [Hành động nếu có]."

Ví dụ:
"Got it. So the login feature is in progress,
 and the main blocker is waiting for the API spec confirmation."
```

### Chuyển sang người tiếp theo

```
"Thank you, [Tên]."
"Next, [Tên tiếp theo], could you please update us on [task]?"
```

### Hỏi client khi cần clarify

```
"We have one point we'd like to clarify."

"Our current understanding is [giải thích].
 Is that correct?"

"Could you confirm the expected behavior for this scenario?"

"Would you prefer we proceed with [option A],
 or should we wait for your confirmation first?"
```

### Xác nhận action items

```
"Let me quickly summarize the action items before we close:"

  "- [Dev team / Tên cụ thể] will [task] by [deadline]."
  "- [Client] will confirm [điều cần confirm] by [deadline]."
  "- We will follow up on [item] in tomorrow's meeting."

"Does that sound correct to everyone?"
```

### Kết thúc

```
"Thank you everyone. That's all from our side today."

"We'll proceed with the agreed action items
 and provide an update in tomorrow's meeting."

"Have a productive day!"
```

## Tips thực chiến

```
NHỚ 5 QUY TẮC KHI LÀM FACILITATOR:

  ┌───┬──────────────────────────────────────────────────────┐
  │ 1 │ Luôn gọi TÊN + TASK trước khi mời người nói         │
  │   │   ✗ "Ai có update không?"                            │
  │   │   ✓ "John, update on task A?"                        │
  ├───┼──────────────────────────────────────────────────────┤
  │ 2 │ Sau mỗi người: TÓM 1 CÂU rồi mới chuyển người tiếp  │
  │   │   → Giữ mọi người aligned                            │
  │   │   → Tránh hiểu nhầm tích lũy                         │
  ├───┼──────────────────────────────────────────────────────┤
  │ 3 │ Nếu bị khựng: dùng CÂU BRIDGE ngay                  │
  │   │   "Let's move to the next item."                     │
  │   │   "While we wait, let's continue with..."            │
  ├───┼──────────────────────────────────────────────────────┤
  │ 4 │ Kết thúc luôn phải có ACTION ITEMS rõ ràng           │
  │   │   WHO + WHAT + WHEN → không mơ hồ                    │
  ├───┼──────────────────────────────────────────────────────┤
  │ 5 │ Giữ flow: PROGRESS → BLOCKER → SUPPORT NEEDED        │
  │   │   Không để member "kể chuyện dài"                    │
  └───┴──────────────────────────────────────────────────────┘
```

## Vocabulary chuyên nghiệp hay dùng

| Tình huống    | Câu mẫu                                                              |
| ------------- | -------------------------------------------------------------------- |
| Mở meeting    | "Let's get started." / "Let's kick off our daily."                   |
| Mời nói       | "Could you walk us through..." / "Would you like to update us on..." |
| Hỏi tiến độ   | "What was completed?" / "Where are we on this?"                      |
| Hỏi blocker   | "Any blockers at the moment?" / "Is there anything blocking you?"    |
| Hỏi client    | "Could you help clarify...?" / "Is our understanding correct?"       |
| Tránh silence | "Let me move to the next item." / "We can come back to this."        |
| Tóm tắt       | "Just to summarize..." / "To make sure we're aligned..."             |
| Kết thúc      | "That's all from our side." / "We'll follow up on this."             |

## Một ngày thực hành — Daily Checklist cho Facilitator

```
TRƯỚC MEETING (2 phút):
  □ Biết task của từng người hôm nay là gì?
  □ Có điểm nào cần hỏi client không?
  □ Có action item hôm qua chưa xong không?

TRONG MEETING:
  □ Open → Agenda (30 giây)
  □ Mời đúng tên + task
  □ Hỏi 3 câu: yesterday / today / blocker
  □ Tóm 1 câu sau mỗi người
  □ Transition mượt mà
  □ Hỏi client nếu cần clarify
  □ Confirm action items trước khi đóng

SAU MEETING (optional nhưng tốt):
  □ Note lại action items vào Slack/Jira/Email
  □ Tag đúng người, đúng deadline
```
